CLI Usage

Usage: quick-link-js [OPTIONS]... [FILE]...

Options:

  • --exit-fail-on=[CODES]: Fail with a non-zero exit code if any of these errors are found (default: "all")
  • -output-format=[FORMAT]: Format to print feedback where FORMAT is one of: gnu-like (default if omitted), vim-qflist-json
  • -v, --version: Print version information
  • --vim-file-bufnr=[NUMBER]: Select a vim buffer for outputting feedback
  • -h, --help: Print help message
